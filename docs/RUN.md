# Run

Install dependencies:
```
npm install
```

Start the app:
```
npm run dev
```

Open the screen:
- Main projector: `http://localhost:3000`
- Fallback sandbox: `http://localhost:3000/sandbox`

Authorize Apple Music on the screen once. Then chat in the Discord channel:
- "play get lucky"
- "switch to focus"
- "what's playing"
- "what's next"

For production-ish run:
```
npm start
```
